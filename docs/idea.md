# Behavioral Classification Engine

## Overview
AI-powered tool that analyzes social media content and evaluates its impact on user goals and dopamine consumption.

## Problem
Social media platforms are designed to capture attention and increase dopamine-driven behavior. Users lose focus and struggle to achieve their goals.

## Solution
This tool analyzes screenshots from social media and classifies:

- Content type
- Psychological intention
- Dopamine level
- Goal alignment
- Recommended action

## Example

Input:
User uploads social media screenshot

Output:
Content Type: Passive entertainment  
Intention: Distraction  
Dopamine Level: High  
Impact: Negative  

Recommended Action:
Close social media for 10 minutes
