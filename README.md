# Behavioral Classification Engine

AI-powered tool that analyzes social media content and evaluates its impact on user goals and dopamine consumption.

## Problem
Social media platforms are designed to capture attention and increase dopamine-driven consumption.

## Solution
This tool analyzes screenshots and classifies:
- Content type
- Psychological intention
- Dopamine level
- Goal alignment
- Recommended action

## Features
- Goal selection
- Screenshot upload
- AI analysis
- Dopamine classification
- Behavioral recommendation
